# mdt-django

Depends on folder mdt-flutter:
    Run this commands in project folder:
    `flutter clean`
    `flutter pub get`

# Open 2 terminals:
    1:
        `.\pyvenv\Scripts\activate.bat`
        `cd mdt`
        (
            `pip install -r requirements.txt`
        )
        `python manage.py runserver`
    2:
        `cd mdt-flutter`
        (
            `flutter clean`
            `flutter pub get`
        )
        `flutter run -d chrome`
